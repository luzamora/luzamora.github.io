Proyecto Oliv.ia

Oliv.ia es una aplicación web desarrollada con Streamlit que integra análisis de datos y procesamiento del lenguaje natural (NLP) para ofrecer recomendaciones personalizadas de restaurantes basadas en reseñas reales extraídas del dataset de Yelp.

// VER CÓDIGO

Permite a los usuarios explorar visualmente los datos y obtener sugerencias de restaurantes adaptadas a sus preferencias.


Proyecto de finalización del Curso de Inteligencia Artificial del programa Campus Samsung Innovation en colaboración con la EOI.
